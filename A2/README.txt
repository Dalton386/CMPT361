HOW TO BOOT:
1. run "make" to construct all objects;
2. run "./myrobot" with 7 legal arguments to start animation, a possible input will be "./myrobot 0 0 5 5 2 5 -tv";
3. run "make clean" to clean up.
********************************************
HOW TO CONTROL:
1. when the program is run, enter "1" to control base, "2" as lower arm, and "3" as upper arm;
2. in addition, press "s" to view aside, press "t" to change to top view;
3. press "a" to rotate ccw, and press "d" to rotate cw;
4. press "q" to exit.
